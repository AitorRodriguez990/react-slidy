# Example with multiple Slides

import {Fragment, useState} from 'react'
import { Playground, Props } from 'docz'
import ReactSlidy from '../index.js'
import Number from '../examples/Number'
import '../index.scss'

## infinite prop

Use `infinite` to determine if the slidy has infinite slides
<Playground>
  <ReactSlidy infinite numOfSlides={1}>
    <Number num={1} />
    <Number num={2} />
    <Number num={3} />
  </ReactSlidy>
</Playground>

## multiple slides

Use `numOfSlides` prop to determine the number of slides.
